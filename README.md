# FFXIV_TexTools_CLI


**FFXIV_TexTools_CLI** is a crossplatform CLI alternative to the Windows-Only *FFXIV_TexTools_UI*

Based on the *[xivModdingFramework](https://github.com/liinko/xivModdingFramework)*
